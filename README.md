# 🤟 Sign Language Detection (CNN)

## 🚀 Overview
A computer vision system that recognizes American Sign Language (ASL) gestures using a Convolutional Neural Network (CNN).

## 🧠 Problem Statement
Communication barriers exist for hearing-impaired individuals. This project aims to translate sign language gestures into readable text using deep learning.

## ⚙️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## 🛠 Architecture / Approach
- Collected and preprocessed gesture image dataset
- Built CNN model for multi-class gesture classification
- Trained model on labeled gesture data
- Used OpenCV for real-time image capture and prediction

## 📊 Results
- ✅ Achieved high accuracy in gesture classification
- ✅ Enabled real-time detection using webcam input
- ✅ Demonstrated robust performance across different gestures

## ▶️ How to Run

git clone https://github.com/supreethbmohan27/sign-language-detection-cnn
